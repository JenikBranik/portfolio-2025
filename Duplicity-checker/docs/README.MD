<h1>Duplicate Checker</h1>

## Features

* **Automatic component analysis**: Just enter the path to the destination folder
* **High performance and reliability:** thanks to multiprocessing and Sha256 it effectively checks file duplication

## How to run it?

* Start run.bat and enter the path for the folder where you want to check for duplicates

## Specifications
* You will only need **python 3.8** and above
